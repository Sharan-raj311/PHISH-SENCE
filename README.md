# PHISH-SENCE — Intelligent Detection of Phishing Threats

PHISH-SENCE is a smart, real-time phishing detection system built using modern 
Artificial Intelligence techniques. Unlike traditional rule-based security systems, 
PHISH-SENCE adapts to new and evolving threats by combining multiple detection methods:

- 📧 **Text Analysis** — Examines email/message content using NLP to identify suspicious language patterns
- 🖼️ **Visual Inspection** — Analyzes website layout and structure to detect brand impersonation
- 🕸️ **Network Graph Analysis** — Maps connections between domains and servers to uncover hidden phishing networks
- 🔒 **Sandboxed Isolation** — Isolates and monitors detected threats without impacting system performance
- ⚡ **Real-Time Protection** — Lightweight browser extension and email platform integration for instant alerts

Built as a final year project at Malla Reddy University.
